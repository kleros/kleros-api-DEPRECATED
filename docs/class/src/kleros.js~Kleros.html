    Instantiates a new Kelros instance that provides the public interface to Kleros contracts and library.

    Params:

    NameTypeAttributeDescription
    ethereumProvider string

    The Web3.js Provider instance you would like the Kleros.js library to use for interacting with the Ethereum network.
    storeProvider string

    The storage provider instance used by the contract to get data from the cloud. e.g. Kleros-Store, IPFS, Swarm etc.

    object

    A n instance of the Kleros.js class.
